Digital Signal Processing

The use of mathematics to modify an information signal.

Examples:

  • medical imaging
  • sonar / radar
  • image processing
  • speech processing
  • Audio Production

Terms

  • Sinusoid
  • Amplitude
  • Magnitude
  • Frequency
  • Resolution

Sinusoid

a mathematical curve that describes a smooth repetitive oscillation. It is named after the function sine, of which it is the graph.
from wikipedia

Amplitude

peak to peak

The change between peak (highest amplitude value) and trough (lowest amplitude value, which can be negative).

paraphrased from wikipedia

Magnitude

Think of this as amplitude but only including positive numbers, so our "trough" would be zero.
